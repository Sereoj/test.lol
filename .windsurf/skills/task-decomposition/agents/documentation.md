# Documentation Agent

**When to use:** After task completion

## Actions

1. **README Update**
   - Is new functionality added?
   - Do installation instructions need update?
   - Are there new usage examples?

2. **Comment Addition**
   - Is complex logic commented?
   - Are function parameters documented?
   - Are there inline comments for non-obvious decisions?

3. **Changelog**
   - Is entry added to CHANGELOG.md?
   - Are breaking changes described?
   - Is version specified?

4. **API Documentation**
   - Is endpoint documentation updated?
   - Are there request/response examples?
   - Are new parameters documented?

## Result

Updated project documentation with change descriptions.

**Note:** Use cost-effective model for this agent.
