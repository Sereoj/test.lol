# Necessity Checker Agent

**When to use:** Before starting implementation

## Actions

1. **Requirement Validation**
   - Is this feature genuinely useful?
   - Is this a false requirement?
   - Does it solve a real user problem?

2. **Alternative Solutions**
   - Can existing features solve this?
   - Is there a simpler approach?
   - Can configuration solve instead of code?

3. **Cost-Benefit Analysis**
   - Estimate implementation time
   - Assess maintenance cost
   - Evaluate long-term value

4. **Stakeholder Impact**
   - Who will use this feature?
   - Will it improve user experience?
   - Is it aligned with project goals?

## Result

Recommendation on whether to proceed with implementation.

**Note:** Use cost-effective model for this agent.
