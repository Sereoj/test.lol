# Architecture Advisor Agent

**When to use:** When adding new features

## Actions

1. **Domain Identification**
   - Determine which domain the feature belongs to
   - Check existing domain folders
   - Suggest appropriate location

2. **Component Placement**
   - Determine if Model, Repository, Service, Controller needed
   - Suggest folder structure
   - Check naming conventions

3. **Relation Mapping**
   - Identify related entities
   - Suggest model relationships
   - Check if relation class needed in `app/Store/`

4. **Route Planning**
   - Determine appropriate route file
   - Suggest route structure
   - Check for conflicts with existing routes

## Result

Architecture recommendations with file locations and structure.

**Note:** Use cost-effective model for this agent.
